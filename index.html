Simple CSS Button hover effect

CSS Code

Fill Up

        
/* button styling */
button {
  padding: 10px 20px;
  background: #f8f3ff;
  color: #2D0367;
  font-family: "Quicksand", sans-serif;
  font-size: 1.2em;
  border: 2px solid #2D0367;
  transition: 0.25s;
}
button:hover, button:focus {
  outline: none;
}

/* hover effects */
.fill-up:hover, .fill-up:focus {
  color: #f8f3ff;
  box-shadow: inset 0 -3em 0 0 #2D0367;
}
